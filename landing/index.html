AuraLens is a Chrome Extension that analyzes your portfolio with AI and recommends optimal DeFi strategies.

Get AuraLens for Chrome Watch Demo

Key Features

Portfolio Overview

See your token balances across chains with a clean, real-time portfolio view.

AI Strategies

Get personalized yield opportunities with APY and risk insights, powered by AURA.

Risk Insights

Monitor risks with AI-driven alerts and contextual notes before you take action.

Seamless UX

Works directly inside your wallet popup with a fast, modern, responsive UI.

How It Works

1

Add Wallet Address

Open AuraLens, add or select your EVM wallet address inside the Wallets modal.

2

Analyze with AURA

Aura fetches balances and strategies from AdEx AURA to build insights.

3

See Insights & Act

Review AI strategies with risks, APY, and notes. Take action when ready.

See AuraLens in Action

Installation

  1. Download/clone the AuraLens repository
  2. Open chrome://extensions
  3. Enable Developer Mode
  4. Click "Load unpacked" and select the extension/ folder
